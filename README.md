## 技术栈
前后端分离  
前端：vue + vuex + vue-cli + vue-router + axios + element-ui  
后端：express + nodejs + mongodb  
数据库：mongodb  

### 下载依赖
`npm install`

### 运行
前端：`npm run serve`  
后端：`npm start`
